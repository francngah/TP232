   # 🌿 AgroData 237 - Analyse Expert 2026

## 📋 Présentation du Projet
Ce projet a été développé dans le cadre du **TP INF 232** (Analyse de données) pour l'année 2026. Il s'agit d'une plateforme intelligente de gestion et d'analyse descriptive des données agricoles au Cameroun.

**Réalisé par :** NGAH FRANC
**Université :** Université de Yaoundé I

## 🚀 Fonctionnalités
- **Tableau de Bord Cyber-Agro :** Visualisation en temps réel des KPIs agricoles (Rendement, Surface, Revenus).
- **Collecte de Données :** Formulaire de saisie complet avec validation.
- **Analyses Statistiques :** Tendance centrale, dispersion, tests de normalité (Shapiro-Wilk) et indicateurs de forme (Skewness/Kurtosis).
- **Exploration Graphique :** Histogrammes avec marginales, nuages de points avec lignes de tendance OLS et matrices de corrélation.
- **Exportation :** Génération de rapports au format CSV et Excel.

## 🛠️ Installation et Lancement
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/VOTRE_NOM/AgroData237_TP_2026.git
   ```
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```
3. Lancez l'application :
   ```bash
   streamlit run app.py
   ```
   *Ou utilisez simplement le fichier `Lancer_AgroData.bat` sur Windows.*

## 🧪 Données de Test
L'application inclut un module de chargement de données de démonstration (200 parcelles simulées) pour tester instantanément toutes les fonctionnalités d'analyse.
